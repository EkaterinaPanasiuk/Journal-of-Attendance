# Journal-of-Attendance/ Журнал успеваемости
## About the App/ О приложении:

The application was developed for keeping an electronic journal of the progress of students of the academy. The application provides for the automation of the administrator's work (viewing, adding, editing users).
Приложение разработано для ведения электронного журнала успеваемости учеников академии. Приложение предусматривает автоматизацию работы администратора( просмотр, добавление, редактирование пользователей).

The initial registration is performed by the System Administrator and access to the software is provided after confirming the registered account in the system with the username entered by the User./// Вход в приложение и доступ к функционалу предоставляется пользователям после авторизации в приложении.
